API REST — Prueba de Endpoints

📡

GET /sensor

Retorna las últimas 100 lecturas del sensor LDR.


🔍

GET /sensor/:valor

Filtra lecturas por valor exacto. Ejemplo: /sensor/0 devuelve lecturas de oscuridad total.


💡

GET /led

Consulta el estado actual del LED integrado (GPIO 2).


🟢

GET /led/1/on

Enciende el LED integrado del ESP32 (GPIO 2 HIGH).


🔴

GET /led/0/off

Apaga el LED integrado del ESP32 (GPIO 2 LOW).


📤

POST /sensor

Inserta una lectura manual enviando JSON con valor_ldr.

Acerca del Proyecto

🌐

Descripción del Sistema

Sistema con ESP32 que mide la intensidad lumínica mediante un sensor LDR y permite el monitoreo en tiempo real desde la nube. Los datos se almacenan en Supabase (PostgreSQL) y se visualizan en este dashboard con gráficas interactivas.

🔩

Componentes de Hardware

  • 🔷 ESP32 DevKit
  • 💡 Sensor LDR (GPIO 34)
  • 🔴 LED Rojo (GPIO 13)
  • 🟡 LED Amarillo (GPIO 12)
  • 🟢 LED Verde (GPIO 14)
  • ⚪ LED Integrado (GPIO 2)
☁️

Tecnologías Utilizadas

  • ⚡ ESP32
  • 🗄️ Supabase (PostgreSQL + Realtime)
  • 📊 Plotly.js
  • 📧 EmailJS
  • 🌐 HTML / CSS / JS
  • 🖥️ Node.js + Express
